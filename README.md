# Food

A React Native app built with Expo to search and explore restaurants using the Yelp API.

## Features

- Search bar with live input
- Categorized restaurant lists by price
- Detailed restaurant view with photos

## Setup

1. Clone the repo
2. Run `npm install`
3. Create `src/config/secrets.js` with your Yelp API key:

```js
export const YELP_API_KEY = "your_key_here";
```

4. Start the app with npm start
